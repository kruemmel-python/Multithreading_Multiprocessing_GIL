Program 1: Multithreading und Multiprocessing¶

Dies ist die Dokumentation der Datei Multithreading_Multiprocessing.py.

  1"""
  2Multithreading und Multiprocessing Beispiel in Python 3.13
  3==========================================================
  4
  5Dieses Modul zeigt, wie man Multithreading und Multiprocessing in Python 3.13 kombinieren kann.
  6Es gibt mehrere Prozesse, die jeweils mehrere Threads innerhalb eines Prozesses starten, um Aufgaben parallel auszuführen.
  7
  8Module:
  9-------
 10- `multiprocessing`: Bietet Unterstützung für die parallele Ausführung von Prozessen.
 11- `threading`: Ermöglicht die parallele Ausführung von Threads innerhalb eines Prozesses.
 12
 13Funktionen:
 14-----------
 15- `thread_task(task_id)`: Führt eine simulierte Aufgabe aus, die durch einen Thread erledigt wird.
 16- `process_task(process_id)`: Erstellt mehrere Threads innerhalb eines Prozesses und führt sie aus.
 17- `main`: Erstellt mehrere Prozesse, die wiederum mehrere Threads enthalten, und synchronisiert deren Ausführung.
 18"""
 19
 20import multiprocessing
 21import threading
 22import time
 23
 24def thread_task(task_id):
 25    """
 26    Führt eine Aufgabe in einem separaten Thread aus.
 27    
 28    Jeder Thread simuliert eine Aufgabe, indem er für 2 Sekunden schläft, bevor er seine Arbeit beendet.
 29
 30    Args:
 31    -----
 32    task_id (int): Die ID des Threads, um die Ausgabe zu kennzeichnen.
 33
 34    Beispiel:
 35    ---------
 36    >>> thread_task(1)
 37    Thread 1 startet.
 38    (Nach 2 Sekunden)
 39    Thread 1 beendet.
 40    """
 41    print(f"Thread {task_id} startet.")
 42    time.sleep(2)
 43    print(f"Thread {task_id} beendet.")
 44
 45def process_task(process_id):
 46    """
 47    Erstellt und startet mehrere Threads innerhalb eines Prozesses.
 48    
 49    Jeder Prozess startet 3 Threads, die parallel Aufgaben ausführen.
 50
 51    Args:
 52    -----
 53    process_id (int): Die ID des Prozesses, um die Ausgabe zu kennzeichnen.
 54
 55    Beispiel:
 56    ---------
 57    >>> process_task(1)
 58    Prozess 1 startet.
 59    Thread 0 startet.
 60    Thread 1 startet.
 61    Thread 2 startet.
 62    Thread 0 beendet.
 63    Thread 1 beendet.
 64    Thread 2 beendet.
 65    Prozess 1 beendet.
 66    """
 67    print(f"Prozess {process_id} startet.")
 68    
 69    # Liste zur Speicherung der Threads
 70    threads = []
 71    
 72    # Erstellen und Starten von 3 Threads
 73    for i in range(3):
 74        t = threading.Thread(target=thread_task, args=(i,))
 75        threads.append(t)
 76        t.start()
 77
 78    # Warten auf das Ende aller Threads
 79    for t in threads:
 80        t.join()
 81    
 82    print(f"Prozess {process_id} beendet.")
 83
 84if __name__ == "__main__":
 85    """
 86    Hauptfunktion des Programms.
 87    
 88    Erstellt mehrere Prozesse (in diesem Beispiel 2). Jeder Prozess führt 
 89    eine Funktion `process_task` aus, die wiederum Threads startet. Die 
 90    Prozesse und Threads laufen parallel und werden am Ende synchronisiert.
 91
 92    Funktionsweise:
 93    ---------------
 94    1. Zwei Prozesse werden erstellt.
 95    2. Jeder Prozess startet drei Threads, die Aufgaben parallel ausführen.
 96    3. Das Hauptprogramm wartet darauf, dass alle Prozesse und Threads beendet werden.
 97
 98    Beispiel:
 99    ---------
100    >>> main()
101    Prozess 0 startet.
102    Thread 0 startet.
103    Thread 1 startet.
104    Thread 2 startet.
105    Thread 0 beendet.
106    Thread 1 beendet.
107    Thread 2 beendet.
108    Prozess 0 beendet.
109    Prozess 1 startet.
110    Thread 0 startet.
111    Thread 1 startet.
112    Thread 2 startet.
113    Thread 0 beendet.
114    Thread 1 beendet.
115    Thread 2 beendet.
116    Prozess 1 beendet.
117    Alle Prozesse und Threads beendet.
118    """
119    # Anzahl der Prozesse
120    num_processes = 2
121
122    # Liste zur Speicherung der Prozesse
123    processes = []
124    
125    # Erstellen und Starten von Prozessen
126    for i in range(num_processes):
127        p = multiprocessing.Process(target=process_task, args=(i,))
128        processes.append(p)
129        p.start()
130
131    # Warten auf das Ende aller Prozesse
132    for p in processes:
133        p.join()
134
135    print("Alle Prozesse und Threads beendet.")
